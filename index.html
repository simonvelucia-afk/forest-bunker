    Guide GitHub Pages

    Comment mettre cette page sur GitHub

    1. Créer la structure
    forest-bunker/ ├── index.html └── images/ ├── bunker-diagram.png ├── modulimo-logo.png └── modulimo-qr.png
    2. Ajouter tes images

    Place l’affiche finale dans images/bunker-diagram.png, le logo dans images/modulimo-logo.png et le QR code dans images/modulimo-qr.png.

    3. Pousser sur GitHub
    git init git add . git commit -m "Version interactive" git branch -M main git remote add origin https://github.com/TON-NOM/forest-bunker.git git push -u origin main
    4. Activer GitHub Pages

    Dans Settings → Pages, choisis Deploy from branch, puis main et /root. Ton site sera publié automatiquement.

    Astuce : pour ajuster les zones cliquables, modifie simplement les coordonnées x et y de chaque composante dans la section JavaScript.
